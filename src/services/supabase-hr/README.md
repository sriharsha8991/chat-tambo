This folder contains modular Supabase HR service logic.

- base.ts: shared types and Supabase helpers
- employees.ts: employee queries
- leave-balances.ts: leave balance queries
- leave-requests.ts: leave request workflows
- attendance.ts: attendance queries
- regularizations.ts: regularization workflows
- notifications.ts: notifications and read tracking
- policies.ts: policy search
- metrics.ts: system metrics queries
- subscriptions.ts: realtime subscriptions
